## Color Legend

- **Blue** (#2563EB): Primary buttons, accents, links
- **Light Blue** (#60A5FA): Hover effects, badges
- **White** (#FFFFFF): Cards, buttons
- **Off-White** (#FAFAFA): Background
- **Dark Gray** (#111827): Main text
- **Gray** (#6B7280): Secondary text
- **Light Gray** (#E5E7EB): Borders

## Card Styles

### Bento Card
```
┌─────────────────┐
│                 │  - White background
│   Content       │  - 16px border radius
│                 │  - 1.5px border
│                 │  - Light shadow (0.03 opacity)
└─────────────────┘  - Hover: stronger shadow
```

### Project Card
```
┌─────────────────┐
│ Title           │  - White background
│                 │  - Title + Description
│ Description...  │  - Tech chips
│                 │  - Link buttons
│ [Tech] [Stack]  │  - Hover: blue shadow
│                 │
│ [link] [link]   │
└─────────────────┘
```

### Contact Card
```
┌─────────────────┐
│ [Icon] →        │  - White background
│                 │  - Icon + title + value
│ Title           │  - Hover: blue background
│ Contact Value   │  - Arrow indicator
│                 │  - Clickable
└─────────────────┘
```

## Spacing System

- Section padding: 96px (space12)
- Between sections: 96px (space12)
- Between cards: 24px (space3)
- Card padding: 32px (space4)
- Between elements: 16px (space2)
- Small gaps: 8px (space1)

## Typography Hierarchy

```
Hero Name (Mohamed)      72px  800  Dark
Hero Name Blue (Nosair)  72px  800  Blue
Section Titles           40px  700  Dark
Card Titles             24px  600  Dark
Body Text               18px  400  Gray
Labels/Badges           14px  500  Light Gray
```

## Interactive States

### Hover Effects
- Cards: Shadow increases from 0.03 to 0.08 opacity
- Buttons: Background color changes
- Contact cards: Background tint + border color
- Timeline dots: Scale up + fill with color

### Animations
- Duration: 200ms
- Easing: Default cubic
- Properties: shadow, background, border, scale

## Responsive Breakpoints

### Desktop (> 1024px)
- Full bento grid layout
- 2-column project grid
- Side-by-side contact cards
- Large font sizes

### Tablet (768-1024px)
- Adjusted bento layout
- 2-column grids
- Medium font sizes

### Mobile (< 768px)
- Single column everything
- Stacked cards
- Smaller fonts
- Reduced spacing
