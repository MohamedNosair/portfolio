# Portfolio - Modern Minimalist Design

## نظرة عامة

بورتفوليو احترافي بتصميم Modern Minimalist مع Bento Grid Layout وألوان Blue & White.

## التصميم

### الألوان
- **Primary Blue**: #2563EB - للأزرار والعناصر المهمة
- **Light Blue**: #60A5FA - للتأثيرات التفاعلية
- **Dark Blue**: #1E40AF - للعناصر الداكنة
- **Background**: #FAFAFA - خلفية بيضاء فاتحة
- **Cards**: #FFFFFF - بطاقات بيضاء نقية
- **Borders**: #E5E7EB - حدود رمادية فاتحة

### المكونات

#### 1. Hero Section
- اسم كبير وواضح
- شارة "متاح للعمل"
- إحصائيات سريعة
- أزرار CTA

#### 2. Bento Grid Section
- بطاقات بأحجام مختلفة
- معلومات عن الخبرة والموقع
- إنجازات رئيسية
- تصميم responsive

#### 3. Projects Section
- عرض المشاريع كبطاقات
- تقنيات مستخدمة
- روابط للمشاريع
- تأثيرات hover

#### 4. Skills Section
- مهارات مجمعة حسب الفئة
- تصميم نظيف
- بطاقات منظمة

#### 5. Experience Timeline
- خط زمني عمودي
- ملون حسب الشركة
- نقاط إنجاز
- تأثيرات تفاعلية

#### 6. Contact Section
- بطاقات تواصل كبيرة
- أيقونات واضحة
- قابلة للنقر
- تأثيرات hover

## التشغيل

### تشغيل التطبيق
```bash
flutter run -d chrome
```

### بناء للإنتاج
```bash
flutter build web --release
```

### فحص الكود
```bash
flutter analyze
```
